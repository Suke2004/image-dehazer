Methodology & Research

Project Overview

This application represents a Computer Vision Final Year Project developed by Sukesh Reddy. The primary objective is to implement a robust environmental dehazing system capable of processing static imagery, batch video files, and real-time camera streams without relying on external deep learning checkpoints.

The backend infrastructure utilizes Python and Flask, heavily leveraging the OpenCV framework for fast matrix operations and computer vision mathematical approximations.

The Dark Channel Prior Algorithm

The core logic strictly implements the Dark Channel Prior (DCP) method. In computer vision, the DCP observation suggests that in most non-sky patches of haze-free outdoor images, at least one color channel has some pixels whose intensity is very low and close to zero.

The pipeline executed in this project includes:

  • Airlight Estimation: Approximating the global atmospheric light from the most haze-opaque regions of the image using morphological erosions.
  • Transmission Map Calculation: Computing the boundary constraints and applying a Kirsch Filters bank to solve for the regularized transmission map.
  • Scene Radiance Recovery: Inverting the atmospheric scattering model formulation to reconstruct the original geometry and contrast.

This implementation successfully demonstrates the applicability of traditional physical scattering models in digital image processing.
