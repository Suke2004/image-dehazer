About the Project

What is the Dehazer App?

The Dehazer App is a smart web-based application designed to remove haze from images, videos, and real-time camera streams. Built with Flask and OpenCV, this project applies image processing techniques — and optionally, deep learning — to restore contrast, enhance clarity, and improve visual quality in low-visibility conditions like fog, smoke, or pollution.

Users can upload hazy media or activate their webcam to instantly see dehazed results through an intuitive, mobile-friendly interface. The app is particularly useful in fields like photography, surveillance, autonomous driving, and environmental monitoring.

Core Features
  • 📸 Dehaze uploaded images with one click
  • 🎞️ Process videos and view side-by-side results
  • 🎥 Real-time dehazing using your device's camera
  • ⚡ Responsive, premium Glassmorphism UI
  • 🛠️ Built with Flask, Python, and OpenCV

The project demonstrates how modern web technologies and computer vision can be combined to solve real-world problems in creative and accessible ways.
